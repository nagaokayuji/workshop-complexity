計算量について

解答

  • 外側のループはN回
  • 内側のループは各 i によって回数が変わる
    • i=1 のとき、j は 1 ~ N までの N 通り
    • i=2 のとき、j は 2, 4, ..., (Nを超えない最大の2の倍数) の個数 ＝ floor(N/2) 通り
    • i=3 のとき、j は 3, 6, ..., (Nを超えない最大の3の倍数) の個数 ＝ floor(N/3) 通り
    • ...
    • i=N のとき j は N のみとなり、1 通り

全部足していくと、∑i=1N​⌊iN​⌋

床関数を外して計算すると ⌊i=1∑N​iN​⌋≤i=1∑N​iN​=Ni=1∑N​i1​

となります。

ここで、 ∑i=1N​i1​について

∫1N+1​x1​dx=log(N+1)≤i=1∑N​i1​(1)

(i=1∑N​i1​)−1=21​+31​+⋯+N1​≤∫1N​x1​dx=logN(2)

が成り立つため、 (1),(2) より log(N+1)≤i=1∑N​i1​≤1+logN が成り立ちます。

N>0 であるから、 Nlog(N+1)≤Ni=1∑N​i1​≤N(1+logN)

よって、計算量は

O(NlogN)

となります。

また、下からも抑えられているため Ω(NlogN) が言えます。
